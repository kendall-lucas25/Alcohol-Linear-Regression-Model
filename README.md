# Alcohol-Linear-Regression-Model
Results: Binge Drinking vs Cancer Prevalence

To evaluate the relationship between binge drinking and cancer prevalence, I performed a linear regression using U.S. state-level data from the Chronic Disease Indicators dataset.

After filtering the data to include only crude prevalence values and aggregating by state, a scatter plot and best-fit regression line were generated.

The results showed a very weak (near-zero) linear relationship between binge drinking prevalence and cancer prevalence within the same year. In some cases, the trend appeared slightly negative, but overall the slope of the regression line was close to zero.

This suggests that higher levels of binge drinking in a given year do not correspond to higher cancer prevalence in that same year.

This outcome is expected for several reasons:

Cancer typically develops over long periods of time (often decades), while binge drinking is measured as a short-term behavior.
The dataset reflects prevalence, not incidence, meaning it captures existing cases rather than new ones.
There are many confounding variables that influence cancer rates, such as age distribution, genetics, environmental factors, and access to healthcare.

Overall, the analysis indicates that there is no strong immediate correlation between binge drinking and cancer prevalence when analyzed at the state level for a single year.
